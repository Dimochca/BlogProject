# 📝 Мой блог

ASP.NET Core MVC блог с Web API, системой ролей, правами доступа, цветовой системой ролей и логированием.

## 🚀 Функциональность

### Веб-приложение (MVC)
- 🔐 **Аутентификация и авторизация** – кастомная система с ролями
- 👥 **Динамические роли** – создание, редактирование, назначение ролей с приоритетами
- 🔑 **Права доступа** – через `HasPermission` и динамические роли
- 📝 **Статьи, теги, комментарии** – полный CRUD
- 🔍 **Поиск по тегам и тексту**
- 👁️ **Система просмотров** – топ-3 популярных статей
- 👤 **Профиль пользователя** – редактирование с ограничением 3 попытки за 5 минут
- 🌐 **Публичный просмотр пользователей**
- 🎨 **Цвета ролей** – везде, где отображается имя пользователя
- 📋 **Логирование** – через NLog всех действий и ошибок
- ⚡ **Глобальный обработчик ошибок**
- ✅ **Валидация форм**
- 🚫 **Страницы ошибок 403 и 404**
- 📱 **Адаптивный дизайн с Bootstrap 5**

### Web API (REST)
- 🔓 **Публичные эндпоинты** – просмотр статей, тегов, пользователей
- 🔐 **Защищённые эндпоинты** – создание, редактирование, удаление (требуется авторизация)
- 📚 **Swagger документация** – интерактивная документация API
- 🔑 **Аутентификация через Cookie**
- 👥 **Управление пользователями, ролями, статьями, тегами, комментариями**

## 🛠️ Технологии

- ASP.NET Core MVC (.NET 8)
- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- SQLite
- Bootstrap 5
- NLog
- Swagger/OpenAPI
- BCrypt.Net
